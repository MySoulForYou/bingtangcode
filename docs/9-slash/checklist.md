# 命令系统验收检查单 (checklist.md)

## 自动化测试与启动期校验项
- [ ] **别名防撞校验**：在测试用例中注册两个相同别名的命令，必须抛出 `IllegalStateException` 异常且系统拒绝启动。
- [ ] **命令解析正确性**：输入 `/PLAN some args` 时，解析器输出的命令名为小写 `plan`，参数为 `some args`。
- [ ] **防误触拦截**：在测试或运行时直接按回车输入空行，系统静默刷新输入框，不得触发大模型 API 调用或生成交互日志。
- [ ] **未知指令异常提示**：输入 `/unknown` 时，控制台输出 `未知命令: unknown, 输入 /help 查看可用命令`（格式需完全匹配）。

## 内置 12 个命令行为验收项
- [ ] **`/plan`**：输入 `/plan` 后，系统的权限模式切换为 `PLAN` 模式，且终端 Prompt 左侧高亮显示 `Plan` 标签。
- [ ] **`/do`**：输入 `/do` 后，系统的权限模式切换为 `DEFAULT` 模式，且 Prompt 标签恢复默认。
- [ ] **`/compact`**：输入 `/compact`，控制台打印出当前估算 Token 及手动压缩阈值，并触发历史消息归档压缩。
- [ ] **`/permission`**：输入 `/permission`，控制台仅按行输出当前权限模式名称（如 `plan`）。
- [ ] **`/session`**：输入 `/session`，控制台输出 `会话ID: [当前session_id]` 和 `存档路径: [绝对路径]`。
- [ ] **`/status`**：输入 `/status` 后，控制台以 `Key: Value` 两列对齐格式，依次输出以下六行且值正确：
  - `Mode:       [当前模式]`
  - `Tokens:     in: [数字] / out: [数字]`
  - `Tools:      [数字]`
  - `Memories:   [数字]`
  - `Model:      [模型名]`
  - `Directory:  [工作路径]`
- [ ] **`/memory`**：输入 `/memory`，若未加载记忆文件，控制台输出 `无已加载的记忆文件`；若有，则按行逐个打印加载的 `.md` 记忆文件名（不读取 `MEMORY.md` 原始文本）。
- [ ] **`/review`**：输入 `/review`，系统在对话历史追加 `请审查上下文中的代码变更，指出潜在 bug、可读性问题、可简化处。` 消息，并自动触发大模型 API 回合，无多余本地逻辑。
- [ ] **`/clear`**：输入 `/clear`，控制台视觉清屏，且主流程关闭当前 JSONL 会话存档文件、开辟新会话存档文件、清空内存消息列表、将 session 累计 token 及 roundCount 计数清零。
- [ ] **`/resume`**：输入 `/resume`，终端清屏并在下方呈现交互式历史会话列表（ActionListBox），用户按 `UP`/`DOWN` 键能改变高亮条目，按 `ENTER` 键确认后会载入选中的历史会话。
- [ ] **`/exit` / `/quit`**：输入 `/exit` 或 `/quit` 后，系统清理底层资源并正常退出程序。
- [ ] **`/help`**：输入 `/help`，终端成功打印帮助卡片，包含这 12 个命令的说明。

## Tab 补全验收项
- [ ] **单匹配补全**：输入 `/comp` 并按 `Tab` 键，输入框内容瞬间自动填满为 `/compact`，无需额外回车。
- [ ] **多匹配提示**：输入 `/p` 并按 `Tab` 键，输入框下方弹出候选菜单，候选列表包含 `/plan` 和 `/permission`，且能通过上下方向键切换。
- [ ] **隐藏命令过滤**：将测试命令的 `hidden` 设为 `true` 后，输入其前缀并按 `Tab` 键，候选列表中不得出现该命令。

## 端到端集成测试项
- [ ] **空斜杠防溢出测试**：在输入框仅输入 `/` 并按回车，控制台不得报错，必须直接列出所有的可用命令列表（效果与 `/help` 相同），且不消耗大模型 Token。
